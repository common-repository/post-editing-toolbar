== Description ==

This plugin implements the toolbar for more comfortable post controls. You can enable/disable it from "Screen Options" panel.

Also you can use hotkeys:

*   Save/update post - Ctrl + S
*   Delete post (move to trash) - Ctrl + D
*   Change status of post - Ctrl + 1
*   Change visibility of post - Ctrl + 2
*   Show revisions - Ctrl + 3
*   Change data of post - Ctrl + 4

== Installation ==

1. Upload `post-editing-toolbar` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
